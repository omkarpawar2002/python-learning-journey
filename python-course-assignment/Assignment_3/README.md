# Assignment-3: Functions & Modules in Python

## Task 1: Calculate Factorial Using a Function
- Takes a number as input from the user.
- Uses a function named `factorial()` to calculate the factorial.
- Uses a loop to perform the calculation.
- Displays the final result.

## Task 2: Using the Math Module for Calculations
- Takes a number as input from the user.
- Uses the `math` module for calculations.
- Calculates the square root of the number.
- Calculates the natural logarithm (log base e) of the number.
- Calculates the sine of the number (in radians).
- Displays all calculated results.

## How to Run
1. Run Task 1:
python task1.py

2. Run Task 2:
python task2.py
